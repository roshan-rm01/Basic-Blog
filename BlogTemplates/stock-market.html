Stock Market Simulator

Programmed using Java

The stock market simulator allows the user to buy or sell any stock, commodity, cryptocurrency or currency. A graphical user interface is used to visually show the user all the data relating with the financial instrument they want to buy and a number of other options. One such option would be to see there portfolio of financial securities while another option would show the latest headlines regarding the stock market. The user can also save there progress and import there data in a later session.

To get real-time stock market data, I used the Twelve data API and CNBC API. The Twelve data API was used to get the current stock, currency and cryptocurrency prices while the CNBC API was used to get the latest headlines and company information. For the commodity data I developed a simple python script to download the latest commodity prices from Yahoo Finance and save it in a specific location. I then used Java to read the file and extract the information. 

Java Swing was mainly used to create the GUI but in the future the simulation will eventually use JavaFX as the main interface. The stock market simulation has 4 main options, trading, portfolio, headlines and quit simulation. For the trading option the user can choose to either buy or sell a security and the market price would be displayed. Other options like portfolio allows the user to browse through there securities, view there transactions and look at there gains and losses. The headlines option will display the latest news headlines and relevant information for a company or country.
